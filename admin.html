Step 1 — Create the Google Sheet
Setup InstructionsFREE · Google Account Required
1
Go to sheets.google.com → click Blank spreadsheet → name it "AAHL 2026-27"
2
Create 7 tabs at the bottom — right-click Sheet1 → Rename. Add tabs by clicking the + button.
3
Go to Extensions → Apps Script. Delete the default code and paste the full script from Step 2 below.
4
In Apps Script: click Triggers (clock icon) → Add Trigger → fetchNightlyStats → Time-driven → Day timer → 3am–4am EST.
5
Back in Sheets: File → Share → Publish to web → Select the Standings tab → Publish. Copy the URL — that's your live public standings link.
6
Paste your Sheet URL into the portal's entry form action and the Admin manual entry section so submissions route correctly.
